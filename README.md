This repository contains all the Rscripts needed to run the analyses underlying the article 
"Evaluating the efficacy of closing the current European network of marine protected areas to mobile bottom-contacting fishing gears",
by KJ van der Reijden, S Valanko, J Depestele, J Desmidt, JM González Irusta, JG Hiddink, P Laffargue, M Sciberras & PD van Denderen,
published by Biological Conservation (DOI:)

It contains 5 consecutive scripts (S1-5) and 5 other scripts that are sourced within these scripts.
All data required for the analyses are publicle available, described in the article, and linked to within the Rscripts.

The scripts themself are published under the MIT-license.
